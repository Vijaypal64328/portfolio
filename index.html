Tic Tac Toe

This is a simple web-based Tic Tac Toe game built using HTML, CSS, and JavaScript. The game features a clean and responsive user interface, allowing two players to take turns marking X and O on a 3x3 grid. JavaScript handles the game logic, including turn switching, win/draw detection, and game reset functionality. It's a fun and interactive project that showcases basic web development skills and DOM manipulation. Click here to PLAY

Weather App

This Weather Application is built using HTML, CSS, and JavaScript. It allows users to search for real-time weather information of any city using a clean and responsive interface. The app fetches data from a weather API and displays key details like temperature, weather conditions, humidity, and wind speed.
